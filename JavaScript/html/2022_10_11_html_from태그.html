from태그의 동작 방법









1.폼이 있는 웹 페이지를 방문
2.폼 내용을 입력한다.
3.폼 안에 있는 모든 데이터를 웹 서버로 보냄
4.웹 서버는 받은 폼 데이터를 처리하기 위해 웹 프로그램으로 넘김
5.웹 프로그램은 폼 데이터를 처리
6.처리결과에 따른 새로운 html 페이지를 웹 서버에 보냅니다.
7.웹 서버는 받은 thml페이지를 브라우저(눈에 보이는 화면)에 보냄
8.브라우저는 받은 html페이지를 보여줍니다.

폼 태그의 속성

폼 태그 속성에는 name, action, methhod, target등이 있다.

action : 폼을 전송할 서버쪽 스크립트 파일을 지정 ex>conroller
name : 폼을 식별하기 위한 이름을 지정 vo, dto와 맞춰주고 getter, setter과 맞춘다.
accept-charset : 폼 전송에 사용 될 문자 인코딩을 지정
targer : action에서 지정한 스크립트 파일을 현재 창이 아닌 다른 위치에 열도록 지정
metgod : 폼을 전송할 http 메소드 정의 GET, POST 두 방식
